# Stonni Atacado — CRM

CRM operacional para o canal de distribuição Bononi Acessórios.

## Stack
- HTML/JS puro (sem framework)
- Supabase Auth + PostgreSQL
- Deploy: Vercel

## Deploy
```bash
git add . && git commit -m "feat: descrição" && git push
```
Vercel deploya automaticamente.

## Acesso
- Módulo: `atacado`
- Hub: https://bononi-hub.vercel.app
